# PyTaskManager

Um gerenciador de tarefas simples em Python para terminal.

## Funcionalidades
- Adicionar tarefas
- Visualizar lista de tarefas
- Atualizar o nome de uma tarefa
- Marcar tarefa como completa
- Deletar tarefa por índice
- Deletar tarefa completada (uma por vez)
- Validação de entradas e feedback ao usuário

## Como usar
1. Clone o repositório:
   ```bash
   git clone https://github.com/mrc-augusto/PyTaskManager_v.1.0.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd PyTaskManager_v.1.0
   ```
3. Execute o gerenciador:
   ```bash
   python gerenciador.py
   ```

## Menu
- 1. Adicionar Tarefa
- 2. Ver Tarefas
- 3. Atualizar Tarefa
- 4. Completar Tarefa
- 5. Deletar Tarefa
- 6. Deletar Tarefa Completada
- 7. Sair

## Observações
- As tarefas não são salvas após fechar o programa (não há persistência).
- Para remover todas as tarefas completadas, repita a opção 6 até não restar nenhuma.
